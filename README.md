# Cricket Machine Learning - Elite Performance Classification

## Overview
This project uses machine learning models to classify elite batting performance across Test, ODI, and T20 formats.

## Dataset
- 13,661 player records
- 17 features including average, strike rate, and engineered features

## Models Used
- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)

## Key Results
- Random Forest performed best
- High F1-score and recall
- Hyperparameter tuning improved performance

## Techniques Used
- Data preprocessing
- Feature engineering
- Model evaluation (Accuracy, Precision, Recall, F1-score)
- Hyperparameter tuning (GridSearchCV)

## Conclusion
Machine learning can effectively classify elite players using multiple performance metrics.
